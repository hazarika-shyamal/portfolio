Secure Chat

secure chat

project that implements a secure server capable of handling multiple modes of communication. The server is written in Python and uses the SSL/TLS protocol to encrypt all communication with clients. The project allows clients to switch between three modes of communication: chat, file transfer, and shell access. In chat mode, clients can send simple text messages that are received and displayed by the server. In file transfer mode, clients can upload compressed files to the server, which the server then decompresses, computes a hash value, and saves to disk. The server sends the hash value back to the client for verification. In shell mode, clients can execute shell commands on the server and receive the output of those commands. The server implementation uses the Python socket module for network communication, the ssl module for SSL/TLS encryption, and the subprocess module for executing shell commands. The project is multi-threaded, with each client connection handled by a separate thread. Overall, the project demonstrates how to implement a secure server capable of handling multiple modes of communication in Python using the SSL/TLS protocol.

WebM

Web Monitor

This project implements a webpage monitoring tool using Selenium, BeautifulSoup, and Matplotlib libraries. It defines a Monitor class that takes a webpage URL, a specific HTML tag, its attribute, and value to monitor the changes in the page content at regular intervals. If a change occurs, it sends an email notification to the specified email address with the previous and current states of the page. Additionally, it plots a graph of the changes over time. The program uses Chrome WebDriver to load the webpage, extracts the specified HTML element using BeautifulSoup, and compares its previous and current states. It sends an email using the Simple Mail Transfer Protocol (SMTP) and plots a graph using Matplotlib. The code is modular, object-oriented, and can monitor multiple webpages simultaneously.

Web Scraper

Scrapper

This project involves extracting web page content from given URLs, using the Python libraries BeautifulSoup, requests, and json. The extracted content is analyzed using natural language processing tools from the nltk library, including tokenization and stopword removal. The resulting analysis includes tag counts and the top 10 keywords for each web page, which are plotted using the pandas and matplotlib libraries. The project also involves extracting all links from the web page and saving them in a text file. Additionally, the project employs networkx library for constructing a graph to visualize the interlinkages between different pages. This tool can be useful for website analysis and optimization. Overall, this project provides a comprehensive analysis of web page content, allowing for a deeper understanding of the information contained within a website.

Senan

sentiment analysis on amazon dataset

The project is a sentiment analysis on Amazon product reviews. The user inputs the ASIN ID of a product on Amazon and the program uses web scraping to retrieve reviews for that product. The reviews are then analyzed using a Naive Bayes algorithm to determine whether they are positive or negative. The program then displays the product along with a graph showing the distribution of positive and negative reviews. The purpose of this project is to provide a tool that can help businesses gather insights into customer feedback and improve their products accordingly. It demonstrates proficiency in web scraping, data analysis, and data visualization using Python.
